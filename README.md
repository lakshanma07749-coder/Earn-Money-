# Earn Money - Make Money Online

A web-based earning platform where users can type numbers, complete tasks, and earn real money.

## Features

- User Registration & Login
- Type & Earn Game (1000 submits = Rs 100)
- YouTube Subscribe & Earn (Rs 10 per subscribe)
- Balance Tracking & Withdrawal System
- Ad Integration

## Pages

- Home
- Type & Earn
- YouTube
- Withdraw

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to Settings → Pages
4. Select "Deploy from a branch" and choose "main"
5. Your site will be live at `https://yourusername.github.io/repository-name`
